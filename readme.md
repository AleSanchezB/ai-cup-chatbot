# Classfication Images Using Siamse-Network

Project AI Cup TE Connectivity

## Install Requirements

```bash
python -m venv .env
# Para Linux/Mac
source .env/bin/activate
# Para windows
.env\Script\Activate
pip install requirements.txt
```

## Run Project

```bash
python main.py
```
